1. Taylor expansion¶

$ f(x)=\sum_{n=0}^\infty \frac{1}{n!}f^{(n)}(x_0)\,(x-x_0)^n$

The Taylor series is a series which can calculate the entire range of any continuous function. This of course implies that all the values of said function are known for all $x$.

We use the exponential function $ f^{(n)} $ and $ x_0 $, the argument around which we expand the function, to define the formula above.

1.1 The exponential function¶

$ f(x)=f'(x)=f''(x)=\dots=f^{(n)}(x)=\exp(x) $

Here, we can define a basic function used by the Taylor expantion formula at the top. We can use this to iterate through all $x$ values of a suitable function, and retrieve the corresponding $y$ values.

In [2]:
def taylor_exp(x, x_0, n_max):
    # x: the argument
    # x0: the argument at which the nth derivatives will be calculated
    # n_max: the itterator and series termination/upperbound
    y = 0
    for n in range(n_max + 1):
        # np.exp(x_0) is a build in function to calculate the nth derivative of a function
        y = y + np.exp(x_0)*(x - x_0)**n / np.math.factorial(n)
    return y;

Using this function we can find the value of Euler's number, which is typically found using this formula:

$e=\sum_{n=0}^\infty \frac{1}{n!}$

Set the upper bound (as computers cannot run until infinity) to 10.

1.2 Maclaurin Expansion at $sin(x)$, at $x_0=0$¶

The Maxlaurin series is a special case of the Taylor expantion series, where $x_0=0$; a single point.

The derivatives of the two trigonometric functions $sin(x)$ and $cos(x)$ are as follows:

$sin(x) = 0$

$sin(x)' = cos(x)$

$sin(x)'' = cos(x)' = -sin(x)$

$sin(x)''' = cos(x)'' = -sin(x)' = -cos(x)$

$sin(x)'''' = cos(x)''' = -sin(x)'' = -cos(x)' = -sin(x)$

While this may look like a mess, all it shows is when continuously deriving these two functions, eventually we start where we began. It's a loop, and this makes sense. The reason it makes sense is due to the fact $sin(x)$ and $cos(x)$ outputs range from -1 to 1.

Another explanation of the process, where only replacing $sin(x)$ with $f(x)$:

$ f(0) = f''(0) = f^{(4)}(0) = \dots = 0 $

$ f'(0) = f^{(5)}(0) = f^{(9)}(0) = \dots = 1 $

$ f'''(0) = f^{(7)}(0) = f^{(11)}(0) = \dots = -1 $

Here is the mathematical definition of the process:

$ \sin(x) = x - \frac{1}{3!}x^3 + \frac{1}{5!}x^5 - \frac{1}{7!}x^7 \pm \dots = \sum_{n=0}^\infty \frac{(-1)^n}{(2n+1)!}x^{2n+1}$

This acheives the same desired outcome as the Taylor expanion near the beggining of the report; only now it is focused on the function $sin(x)$ using Maclaurin's expantion. The $y$ value outputs should match the $y$ value outputs of the $sin$ function, within a specified range.

In [5]:
def taylor_exp_sin(x, n_max):
    # x: the argument
    # n_max: the itterator and series termination/upperbound
    y = 0
    for n in range(n_max + 1):
        y = y + (-1)**n * x**(2*n + 1) / np.math.factorial(2 * n + 1)
    return y

Notice the absence of the exp function in the formula. It is not required since the outputs from $f(x)^{(n)}$, $f(x)$ being $sin(x)$, loop infinitly within the range of -1 to 1.

1.3 Implementing a general purpose Taylor series function¶

Before a general Taylor series function can be deployed, a function to find derivatives must be defined.

The formula to calculate the derivative of a function: $ f'(x) = \lim_{h\rightarrow 0} \frac{f(x+h)-f(x)}{h} $

In [8]:
def derivative(f, x, h):
    # f: the function
    # x: argument of f
    # h: step size
    return ((f(x + h) - f(x)) / h)

A function to calculate the $nth$ derivative of a function can also be defined.

The formula to calculate higher derivatives: $ f^{(n)}(x) = \lim_{h\rightarrow 0} \frac{1}{h^n}\sum_{k=0}^n(-1)^{k+n} \,\frac{n!}{k!(n-k)!} \,f(x+kh)$

In [9]:
def nDerivative(f, x, h, n):
    # f: the function
    # x: argument of f
    # h: step size
    # n: nth derivative
    y = 0
    for k in range(n+1):
        y = y + ((-1)**(k + n)*(np.math.factorial(n)*f(x + k * h))) / (np.math.factorial(k)*np.math.factorial(n - k))
    return y / h**n

1.4 Implementing a general Taylor expansion function using the $nth$ derivatives function¶

In [15]:
def taylor(f, x, x0, n_max, h): # lets define it for all(most) continuous functions
    # f   : the function
    # x   : argument
    # x0  : argument at which the derivative will be calculated
    # n_max: upperbound for series termination (sum upperbound)
    # h   : Stepsize
    y = 0
    for n in range(n_max + 1):
        y = y + (nDerivative(f, x0, h, n) * (x - x0)**n) / np.math.factorial(n)
    return y

2.3.1 Define model function¶

For practice, we consider a polynomial: $ f(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_n x^n = \sum_{k=0}^n a_k x^k$

def polynomialModel(x, a):
    # x: the argument of the function
    # a_n: the coefficients
    y = 0
    for k in range(len(a)):
        y = y + a[k] * x**k
    return y

2.3.2 Defining error¶

The error function used here: $ \Delta = \sum_{i=1}^n \left(y_i-f(x_i)\right)^2$

This is a commonly used error funciton.

$ f(x_i)$ is the fit function that is determined by all coefficients $ a_k $ in the n-1 order polynomial model function, defined in subsection 2.3.1.

The points to fit are the purturbed data's $ x_i$ and $y_i$, and here the sum of the error will be determined.

In [40]:
def errorFit(f, coefficients, data):
    # f: the fit function
    # coefficients: a_k -> the model function coefficients used to optomize
    # data: the purturbed data to fit
    error = 0
    for i in range(len(data[0])):
        error = error + (data[1,i] - f(data[0,i], coefficients))**2
    return error

2.3.3 Gradient descent: Updating coefficients to reduce the error at each point¶

Here, we will use the gradient descent method to update the coefficients $ a_k $. These coefficients will be updated along the gradient direction of the error function (nabla) $ \nabla_{\vec{a}} \Delta$.

Here, $\Delta$ is the error fit function defined in subsection 2.3.2.

The mathematical deffinition is defined as the derivative of $\Delta$ with respect to $a_k$:

$ \frac{\partial}{\partial a_k} \Delta = -2 \sum_{i=1}^n \left(y_i-f(x_i)\right) \frac{\partial}{\partial a_k}f(x_i) = -2 \sum_{i=1}^n \left(y_i-f(x_i)\right) x_i^{k}$

The Monte-Carlo algorithm could also be used if desired.

In [42]:
def errorFitGradient(f, coefficients, data):
    # f: the error fit function
    # coefficients: a_k -> the model function coefficients used to optomize
    # data: the purturbed data to fit
    return -2 * np.array([
        np.sum(
            np.array([(data[1,i] - f(data[0,i], coefficients)) * data[0,i]**k
            for i in range(len(data[0]))]
                    )
              )
        for k in range(len(coefficients))]
    )
